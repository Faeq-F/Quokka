The app comes as portable by design - there are no separate build instructions for a non-portable version.
  1. Open Visual Studio & clone Quokka
  2. Build the solution
  3. Copy the Quokka folder (in build path - normally 'bin') to your desired location (e.g. USB drive)
  4. (Download / Delete) any plugins you (do / do not) wish to use (In the PlugBoard folder)
    (Some of the recognized Plugins may be included with the build)
