The tray task context menu can be used to:
  • Launch the search window
  • Open the settings file
  • Open the PlugBoard
  • Exit the app
To change the tray task icon, edit the 'QuokkaTray.ico' file in '...\Quokka\Config\Resources\'
(or just rename a different .ico file to the same name)
The tray task context menu can have its appearance changed in the settings file. You will have to exit and reload the application to see changes.
You can only have one search window open at a time
You can change which file manager opens the plugboard folder through the FileManager setting
Likewise, you can change which text editor opens the settings file through the TextEditor setting
Unfortunately system tray tasks cannot be accessed through the Quokka search window (though Quokka's tray task can be); this means that Quokka cannot completely replace your taskbar yet.
This functionality may come in future (as a plugin).
