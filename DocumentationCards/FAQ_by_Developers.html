Q:
How do I create a task for the program in Task scheduler?
A:
Follow a guide to create a basic task, like the following guide to create an automated task by Mauro Huculak on Windows Central

You will want to set it to start 'When I log on' and give 'Quokka.exe' (located in the root of the program folder) as the program. You do not need to add any arguments.
