You must install the package (likely through nuget) for your plugins project. Unfortunately this is not enough since Quokka will need the package reference of the library. You will have to use reflection to load the library at runtime and use the classes it provides. You can have a look at the following plugins as examples as to how to do this:

  • Plugin_Calculator
  • Plugin_EnglishDictionary
  • Plugin_Everything
