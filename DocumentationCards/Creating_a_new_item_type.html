Most plugins will want to show results to the user in the form of an item on the results list in the search window. To do so, first create an item type to define how your results are displayed;

  1. In the .cs file add
  2. Create a ListItem class for your item type
    This is a part of the ShowTypedText  plugin (a demo plugin; not made for use)
ListItems must have the following:
  • a Name
  • a Description
  • an Icon
  • an Execute method to define what should happen when a list item is run (i.e., the user hits the enter key when that item is selected)
  • a context pane
The ToString method of ListItems return the item's Name.
