To add a plugin, download and extract it to the PlugBoard folder
If you do not wish to use a plugin, simply delete the appropriate folder in the PlugBoard
Plugins may have their own settings and special commands.
Please see their folder in the PlugBoard to find their settings file (at the root (if there is one)).
Please consult their documentation for default values, etc.
Only recognized plugins appear on this page. Being human, I can't recognize every plugin for Quokka — I can only realize there is so much that exists.
