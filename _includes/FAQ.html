How do I set the app to run at startup automatically?

There are two ways, the second providing slightly more control:

You can create a shortcut to the .exe file of the app (right click to show context menu ➤ click 'Show more options' ➤ click 'Create shortcut') and copy it to the following directory; "C:\Users\USERNAME\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\", where 'USERNAME' is your Windows profile name.

or you can create a task for the program in Task scheduler;
Follow a guide to create a basic task, like the following guide to create an automated task by Mauro Huculak on Windows Central.
You will want to set it to start 'When I log on' and give 'Quokka.exe' (located in the root of the program folder) as the program. You do not need to add any arguments.
