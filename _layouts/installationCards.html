Quokka requires the following:

A computer running Windows 7 or higher
A Windows defender exclusion for the location of the exe

Add exclusions to any other anti-malware software that is installed on your computer;
you do not want the software to mistake the app for viruses or other malware
To install:

Go to the top of this page and click on the 'Download Latest Release' button

Download the 'Quokka.zip' file & extract it's contents

You will want to extract / move the folder to an appropriate location. Quokka is a portable program by design, meaning that you can place the folder on a external storage medium (e.g. a USB drive) and run the program from there. Please see the suggestions below for an appropriate location of the folder.
Suggestions:

If you do not wish to use the program in a portable manner, I suggest placing the folder in 'C:\Program Files', and creating a shortcut that is pinned to the taskbar and / or start menu. You may also want to set up a task to run the program at startup in Task Scheduler
